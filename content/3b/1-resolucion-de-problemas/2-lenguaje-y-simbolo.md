## Lenguaje y símbolo

Existen problemas que para abordarlos requieren de un pasaje símbolico más complejo.

## Ejemplo

> Si al triple de un número le restamos cinco obtenemos el número incrementado en siete unidades. ¿Qué número es?

Para comenzar, conviene **remplazar cada palabra** con su **símbolo matemático** como se ve en la animación:

![Pasaje de simbolo a matemático](../img/simbolico-a-matematico.gif)

## Expresiones útiles

Las siguiente tabla nos será útil para resolver los problemas como el anterior.

#### Más comunes

| Expresión Coloquial                                     | Simbología Matemática |
| ------------------------------------------------------- | :-------------------: |
| Dado un número                                          |          @x@          |
| El duplo, el doble de un número                         |      @2 \cdot x@      |
| Un número disminuido en 13                              |       @x - 13@        |
| El antecesor, o el anterior de un número                |        @x - 1@        |
| El sucesor, el consecuente, o el siguiente de un número |        @x + 1@        |
| El opuesto de un número                                 |         @- x@         |

#### De multiplicación

| Expresión Coloquial       | Simbología Matemática |
| ------------------------- | --------------------- |
| Un número par             | @2 \cdot x@           |
| Número impar              | @2 \cdot x + 1@       |
| El triple de un número    | @3 \cdot x@           |
| El cuádruplo de un número | @4 \cdot x@           |

#### De división

| Expresión Coloquial                        | Simbología Matemática  |
| ------------------------------------------ | :--------------------: |
| La mitad de un número                      | @\dfrac{1}{2} \cdot x@ |
| La tercera parte, o el tercio de un número | @\dfrac{1}{3} \cdot x@ |
| La cuarta parte de un número               | @\dfrac{1}{4} \cdot x@ |
| La quinta parte de un número               | @\dfrac{1}{5} \cdot x@ |

#### De potencia y raíz

| Expresión Coloquial                    | Simbología Matemática |
| -------------------------------------- | :-------------------: |
| El cuadrado de un número               |         @x^2@         |
| El cubo de un número                   |         @x^3@         |
| La raíz cuadrada de un número          |      @\sqrt{x}@       |
| La raíz cubica de un numero            |    @{\sqrt[3]{x}}@    |
| El cuadrado del siguiente de un número |      @(x + 1)^2@      |
| El cubo del siguiente de un número     |      @(x + 1)^3@      |

#### De mención a dos variables

| Expresión Coloquial             | Simbología Matemática |
| ------------------------------- | :-------------------: |
| La razón entre dos números      |    @\dfrac{x}{y}@     |
| El producto entre dos números   |      @x \cdot y@      |
| La diferencia entre dos números |        @x - y@        |

#### De sucesión

| Expresión Coloquial          |                           Simbología Matemática                            |
| ---------------------------- | :------------------------------------------------------------------------: |
| Números consecutivos         |                  @x@; @x + 1@; @x + 2@; @x + 3@; @\dots@                   |
| Números pares consecutivos   |  @2 \cdot x@; @2 \cdot x + 2@; @2 \cdot x + 4@; @2 \cdot x + 6@; @\dots@   |
| Números impares consecutivos | @2 \cdot x + 1@; @2 \cdot x + 3@; @2 \cdot x + 5@; @2\cdot x + 7@; @\dots@ |
