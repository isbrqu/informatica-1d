Definición

Una computadora es un dispositivo electrónico capaz de:

  1. Recibir datos.
  2. Procesar datos.
  3. Almacenar datos.
  4. Enviar datos

Tarea

  1. Escribir una lista de 3 objetos electrónicos que SEAN computadoras.

    • Tablet.
    • Netbook.

  2. Escribir una lista de 3 objetos electrónicos que NO SEAN computadoras.
